# Project Documentation

## Overview
This project, Nuvora-agency, aims to provide a streamlined solution for managing agency operations.

## Features
- Intuitive user interface
- Robust backend services
- Integration with various third-party APIs
- Responsive design for mobile and desktop

## Tech Stack
- Frontend: React, Redux
- Backend: Node.js, Express
- Database: MongoDB
- Hosting: Heroku

## Project Structure
```
Nuvora-agency/
├── client/         # Frontend code
├── server/         # Backend code
├── config/         # Configuration files
└── README.md       # Project documentation
```

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Alaa-Taifour/Nuvora-agency.git
   ```
2. Navigate to the client directory and install dependencies:
   ```bash
   cd client
   npm install
   ```
3. Navigate to the server directory and install dependencies:
   ```bash
   cd server
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## File Descriptions
- `client/`: Contains all frontend code and assets.
- `server/`: Contains backend API services and configurations.
- `config/`: Holds sensitive configurations like database connection details.

## Configuration Files
- `config/default.json`: Default configurations for the application.
- `config/production.json`: Production environment configurations.
